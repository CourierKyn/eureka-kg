# 金融科技知识图谱

金融科技知识图谱平台根据查询的公司、技术，可以找出相关领域的竞争者、上下游技术、潜在的供应商和相关资讯文章等。

## 架构

    .
    ├── README.md # 本文档
    ├── android # 安卓
    ├── crawler # 爬虫
    ├── database # 数据库
    ├── frontend # 前端
    ├── nlp # NLP
    ├── server # 后端
    └── ...
